Detailed Description

A bitfield representing usart1 control and status registers.

Field Documentation

◆ clock_polarity

volatile uint8_t clock_polarity

Rising or falling edge. Use the USART_CLOCK_POL macros. Transmit data is changed on rising edge and rx data sampled on falling by default.
